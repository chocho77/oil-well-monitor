# Oil Well  Monitor

Process monitoring system for oil and gas wells.
This project simulates real-time sensor data from a production well and visualizes it in a dashboard.



## Technologies
     - Java + Spring Boot (Backend)
     - PostgreSQL (Database)
     - Angular (Fronted)
     - Docker (Containerization)


## Features
 - Real-time sensor data simulation (flow rate, pressure, temperature)
 - Interactive dashboard with live charts
 - Alert system for abnormal readings
